How to run:
put cifar10_*.py in one directory.
change line 750 in the saver.py in the library to be "max_to_keep=500".
Your may directly copy my "saver.py" to replace that in the library.

For all *.py, change file routes to appropriate ones.
For problem 1, run python p1.py.
For problem 2, run python p2.py.

For problem 3, in directory p3, run python cifar10_train.py then run python cifar10_eval.py.
For problem 4, in directory p4, run python cifar10_train.py then run python cifar10_eval.py.
